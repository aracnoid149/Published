This is a powerful high performance, low cost micro-controller aimed at the individual who needs capabilities far beyond the 8 and 16 bit worlds yet as simple to use as 8 bit controllers. This controller is fast enough to handle almost anything the user can throw at it.

  1. ARM M7 Core Processor = STM32F753
  2. Core Frequency = 216 MHz with On Board Xtal OSC or With internal OSC 1%
  3. FLASH Size   = 512K
  4. RAM Size      = 256K
  5. Fast Hardware Floating Point Co-Processor
  6. Fast DSP Instruction Set.
  7. Low Power
  8. 18 Timers
  9. 12 Bit ADC & DAC
10. 14 Communications Peripherials
11. On Board 8K x 8 EEPROM
12. Ready to use RTOS system available
13. 54 Fast GPIO I/O Pins, 5 Volt tolerant
14. Runs from +5 VDC Power
15. Compact 60 Pin Package
16. On Board Reset Button
17. On Board Power LED
18. On Board USB Programming
19. On Board Micro USB Connector
