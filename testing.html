This is a powerful light duty project micro-controller unit aimed at the DIY market for users that need a low cost,high performance controller for those projects that require more than the 8 Bit processors can offer. Operating at 80 MHz it leaves the traditional micro-controllers in the dust.

1. ARM M4 Core Processor = STM32L432
2. Core Frequency = 80 MHz
      With internal OSC 1%
      or With On Board Xtal OSC
3. FLASH Size   = 256K
4. RAM Size     = 64K
5. Fast Hardware Floating Point Co-Processor
6. Fast DSP Instruction Set.
7. Very Low Power
8. 11 Timers
9. 12 Bit ADC & DAC
10. 14 Communications Peripherials
11. On Board 8K x 8 EEPROM
12. Ready to use RTOS system included
13. Bootload Programming from Serial Port
14. 26 Fast GPIO I/O Pins
15. Runs from +5 VDC Power
16. Compact 32 Pin DIP Package
17. On Board Reset Button
18. On Board Power LEDNew text box
