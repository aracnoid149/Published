This screaming fast, powerful and performance micro-controller is a low cost micro-controller aimed at the individual who needs capabilities far beyond the typical 8, 16, and 32 bit worlds. Yet this is as simple to use as 8 bit controllers. This controller is fast enough to handle almost anything the user can throw at it. If you can think of it, this can do it.

1. Powerful ARM M7 H Series Core Processor = STM32H743
2. Core Frequency = 480 MHz
      With internal OSC 1%
      or With On Board Xtal OSC
3. FLASH Size   = 2M
4. RAM Size     = 1M
5. Fast Hardware Floating Point Co-Processor
6. Fast DSP Instruction Set.
7. Low Power
8. 20 Timers
9. 12 Bit ADC & DAC
10. 38 Communications Peripherials
11. On Board 8K x 8 EEPROM
12. On Board Graphical Accelerator for fast LCD Interface
13. Ready to use RTOS system included
14. 92 Fast GPIO I/O Pins
15. Runs from +5 VDC Power
16. Compact 60 Pin Package
17. On Board Reset Button
18. On Board Power LED
19. On Board USB Programming
20 Digital Camera Interface
21. LCD-TFT On Board Controller
